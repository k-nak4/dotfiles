# dotfiles

## Requirements
* git

## Install
1. Clone this repository.
2. ```$ cd dotfiles```
3. ```$ <any_text_editor(*1)> install.sh``` and change parameters(*2).
4. ```$ ./install.sh```

*1: ex.```$ nano install.sh```
*2.
```
# Don't install = 0, Install = 1
GIT=0 # Don't install
VIM=1 # Install
NVIM=0
...
BINS=0
```
