# setting-files

## Requirements
* Git

## Optional
* Tmux
* Vimperator
* Conky

## Installation
1. Change directory to ```$HOME```
2. Clone this repository.
3. ```$ cd ~/setting-files; <any_text_editor> install.sh``` and change parameters.
4. Install ```./install.sh```

## Vimperator
1. Run the FireFox.
2. ```:vimppm install```
